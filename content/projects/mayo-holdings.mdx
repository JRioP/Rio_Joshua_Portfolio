## Overview

Full corporate website for MaYo Holdings & Construction, Inc. — built end-to-end
during OJT, from local XAMPP staging to live Namecheap cPanel hosting.

## Key achievement

Custom PHP email routing system using `wpforms_entry_email_to` filter —
maps service type dropdown values to department email addresses with
ticket number assignment and atomic database increment.

## Performance

- 92 PageSpeed score (mobile + desktop)
- A+ external security rating
- LiteSpeed Cache + WebP images + lazy loading
